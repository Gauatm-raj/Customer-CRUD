# Customer-CRUD


## Description

This is a simple CRUD application for managing customers with JWT authentication.

## Technologies Used

- Spring Boot
- MySQL
- HTML/CSS/JS

## How to Run

1. Clone the repository.
2. Configure MySQL database in `application.properties`.
3. Run the Spring Boot application.
4 localhost:8080
